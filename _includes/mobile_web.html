移动web开发像素知识

  • px : css逻辑像素，浏览器使用的抽象单位

  • dp,pt: 设备独立像素

  • dpr: 设备缩放比

计算公式： 1 px = dpr的平方 * dp

  • DPI: 打印机每英寸可以喷的墨汁点
  • PPI：屏幕上每英寸的像素数量

PPI = 根号（物理像素的长^2 + 宽^2） / 4

PPI对应的缩放比参考 移动页面参考模板
